# 🎓 Sistema de Cadastro de Alunos

## 📖 Sobre

Projeto desenvolvido em Python para cadastro e gerenciamento de alunos.

Permite cadastrar estudantes, alterar notas, calcular média da turma e verificar aprovação automaticamente.

Os dados podem ser salvos e carregados utilizando arquivos JSON.

## 🚀 Funcionalidades

- ✅ Adicionar alunos
- ✅ Remover alunos
- ✅ Alterar notas
- ✅ Calcular média da turma
- ✅ Verificar aprovação
- ✅ Salvar dados em JSON
- ✅ Carregar dados em JSON

## 🛠️ Tecnologias

- Python 3
- JSON
